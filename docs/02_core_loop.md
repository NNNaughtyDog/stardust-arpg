# 02. 核心循环

## 主循环

```text
营地整理
↓
出城进入污染区
↓
探索地图与遭遇怪物
↓
击杀怪物获得金币、材料、装备
↓
判断装备是否更强
↓
进入地牢或返回营地
↓
挑战 Boss
↓
获得高价值掉落
↓
回营地整理、强化、再次出发
```

## 15 分钟垂直切片节奏

### 第 0~2 分钟：营地

- 玩家出生在安全营地。
- UI 显示血量、法力、技能栏。
- 简短提示：左键移动/攻击，右键释放技能。

### 第 2~6 分钟：野外

- 玩家进入污染荒野。
- 遇到第一批低威胁怪物。
- 学会走位、攻击、拾取掉落。

### 第 6~10 分钟：精英战

- 出现精英怪或小型怪物群。
- 掉落第一件明显提升的装备。
- 玩家打开背包并装备。

### 第 10~14 分钟：地牢

- 玩家进入 3~5 个房间的小型地牢。
- 怪物密度提高。
- 出现环境阻挡、狭窄通道、拐角遭遇。

### 第 14~15 分钟：Boss

- Boss 拥有 2 个技能。
- 击败后掉落稀有装备。
- 引导玩家回营地。

## 每次战斗的微循环

```text
发现敌人
↓
判断距离
↓
移动/走位
↓
普攻或释放技能
↓
敌人死亡或玩家撤退
↓
拾取掉落
```

## 装备循环

```text
怪物死亡
↓
触发掉落表
↓
生成装备基础类型
↓
判定稀有度
↓
根据稀有度生成词缀
↓
玩家拾取
↓
比较当前装备
↓
装备/出售/分解
```

## 第一版掉落节奏

- 普通怪：少量金币，低概率普通/魔法装备。
- 精英怪：稳定掉落装备，中概率魔法/稀有。
- Boss：稳定掉落稀有装备，低概率传奇。

建议初始权重：

| 稀有度 | 权重 |
| --- | ---: |
| 普通 | 60% |
| 魔法 | 28% |
| 稀有 | 10% |
| 传奇 | 2% |

## 第一版失败条件

- 玩家死亡后回到营地。
- 保留已拾取装备。
- 当前地牢进度可重置。

## 后续扩展循环

后续可以增加：

- 随机地图
- 多层地牢
- 装备分解
- 材料强化
- 词缀重铸
- 符文/宝石
- 职业技能树
- 每日挑战
