# 05. 系统拆解

## 1. 输入与移动系统

### 目标

实现经典 ARPG 的基础操作：

- 左键点地移动。
- 左键点怪攻击。
- 右键释放当前技能。
- Shift + 攻击时原地攻击。

### 第一版验收

- 玩家点击地面后能移动到目标点。
- 到达目标点后停止。
- 玩家不能穿过阻挡碰撞体。
- 移动速度可配置。

## 2. 战斗系统

### 目标

实现最小可用战斗闭环：

- 普通攻击
- 攻击范围
- 攻击冷却
- 伤害结算
- 受击反馈
- 死亡处理

### 第一版公式

```text
最终伤害 = max(1, 攻击力 × 技能倍率 - 防御)
```

## 3. 技能系统

### 初始技能

| 技能 | 类型 | 描述 |
| --- | --- | --- |
| 普通攻击 | 近战 | 基础单体攻击 |
| 冲刺斩 | 位移/伤害 | 向目标方向突进并攻击路径上的敌人 |
| 星尘爆裂 | 范围 | 消耗星尘能量，对周围敌人造成范围伤害 |

### 技能字段

```text
skill_id
name
damage_multiplier
mana_cost
cooldown
range
area_radius
cast_time
hit_effect
```

## 4. 怪物系统

### 初始怪物

| 怪物 | 定位 | 特点 |
| --- | --- | --- |
| 腐化爬虫 | 快速近战 | 低血量，速度快 |
| 荒野暴徒 | 普通近战 | 基础敌人 |
| 星尘射手 | 远程 | 投射物攻击 |
| 精英异化者 | 精英 | 更高血量，更高掉落 |
| 裂隙 Boss | Boss | 大体型，拥有技能前摇 |

### 怪物字段

```text
monster_id
name
level
max_hp
attack
defense
move_speed
aggro_range
attack_range
attack_cooldown
drop_table
```

## 5. 装备系统

### 装备部位

第一版只做：

- 武器
- 头盔
- 护甲
- 戒指

### 稀有度

| 稀有度 | 规则 |
| --- | --- |
| 普通 | 只有基础属性 |
| 魔法 | 1~2 个词缀 |
| 稀有 | 3~5 个词缀 |
| 传奇 | 固定特殊效果 |

### 装备字段

```text
item_id
name
slot
rarity
level
base_stats
affixes
icon_path
special_effect
```

## 6. 词缀系统

### 初始词缀类型

- 攻击力增加
- 防御增加
- 最大生命增加
- 移动速度增加
- 攻击速度增加
- 暴击率增加
- 暴击伤害增加
- 火焰伤害
- 星尘伤害
- 击杀回复生命

## 7. 掉落系统

### 流程

```text
怪物死亡
↓
读取 drop_table
↓
判定是否掉落
↓
判定装备类型
↓
判定稀有度
↓
生成词缀
↓
生成地面掉落物
```

## 8. 背包与装备栏

### 第一版功能

- 拾取装备进入背包。
- 背包用网格或列表显示。
- 点击装备可以穿戴。
- 穿戴后刷新玩家属性。
- 被替换装备回到背包。

第一版 UI：

- 技能栏
- 背包面板

- 怪物血条
