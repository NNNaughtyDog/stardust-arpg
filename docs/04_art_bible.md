# 04. 美术规范 Art Bible

## 总体方向

2D 等距视角、暗黑奇幻、低饱和、高对比、火光与阴影强烈、环境压迫感强。

本项目要学习的是经典等距 ARPG 的镜头语言和氛围组织方式，不使用任何现成游戏的素材、UI、图标、角色、怪物或场景。

## 镜头规则

- 采用 2D 等距/斜 45 度俯视视角。
- 角色在画面中占比较小，突出环境压迫感。
- 摄像机跟随玩家，但不要过度贴近。
- 战斗时允许保留足够视野，让玩家看见怪物群和技能范围。

## 色彩规则

推荐：

- 低饱和棕灰、蓝灰、铁锈红、暗金色。
- 火把、熔岩、星尘污染可作为高亮色。
- 强调冷暖对比：冷色环境 + 暖色火光。

避免：

- 高饱和卡通色。
- 明亮糖果色。
- 现代霓虹赛博感。
- 大面积纯色块。

## 环境关键词

- 破败营地
- 星尘污染荒野
- 风化石墙
- 地下墓穴
- 裂隙洞窟
- 破碎祭坛
- 枯树、荒草、残骸
- 火把、灰烬、符文石

## 角色比例

- 玩家角色应小而清晰。
- 动作剪影要明确：站立、行走、攻击、受击、死亡。
- 不追求大头 Q 版。
- 不走横版卷轴比例。

## 怪物设计原则

第一批怪物优先做剪影差异：

- 低矮爬虫：快速近战。
- 人形暴徒：普通近战。
- 远程射手：投射物攻击。
- 精英异化者：体型更大、带光效。
- 裂隙 Boss：大体型、明显攻击前摇。

## UI 方向

UI 应该厚重、清晰、有材质感，但不能直接模仿任何现成游戏。

推荐材质：

- 暗金属
- 旧皮革
- 石雕边框
- 红/蓝液体球形或半球形资源提示
- 简洁技能栏

避免：

- 直接照搬经典 ARPG UI 排版。
- 过度复杂的装饰。
- 手机页游式满屏按钮。

## 禁止方向

- 不要 Q 版。
- 不要二次元。
- 不要横版卷轴。
- 不要明亮卡通。
- 不要现代科幻霓虹。
- 不要直接照搬任何现成游戏的 UI、怪物、图标或场景。
