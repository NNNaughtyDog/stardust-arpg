# 01. 游戏愿景

## 一句话定位

一款原创 2D 等距视角暗黑奇幻 ARPG：玩家在星尘污染后的废土与地下裂隙中探索、战斗、刷取随机装备，并通过技能与装备组合形成自己的 Build。

## 体验关键词

- 等距俯视
- 黑暗奇幻
- 星尘污染
- 废墟营地
- 地牢探索
- 怪物群战
- 装备驱动成长
- 短周期刷宝循环
- 高反馈战斗

## 玩家幻想

玩家不是传统救世主，而是一个被放逐的幸存者。世界已经被星尘裂隙污染，旧文明坍塌，野外充满异化生物。玩家需要一次次进入污染区，击败怪物，带回战利品，强化自身，逐步接近裂隙源头。

## 设计支柱

### 1. 操作直接

玩家的第一感受必须是：

- 点哪里就往哪里走。
- 点怪物就能打。
- 技能释放清楚、反馈明显。
- 战斗中移动、走位、拉怪有意义。

### 2. 成长明确

每次出门都应该有收获：

- 掉了一件更强的武器。
- 获得一个新词缀组合。
- 技能伤害提升。
- 打怪效率变快。
- 能挑战更深的地牢。

### 3. 氛围压迫

美术方向不是明亮卡通，而是：

- 低饱和色彩
- 冷暖强对比
- 火光、阴影、废墟
- 小比例角色与大面积环境
- 强烈的荒凉、危险、未知感

### 4. 内容可扩展

所有核心内容尽量数据驱动：

- 怪物表
- 技能表
- 装备表
- 词缀表
- 掉落表
- 地图配置

这样 Codex 和其他 AI Agent 后续可以持续追加内容，不需要每次改大量硬编码。

## 原创化原则

本项目禁止直接复刻任何现成游戏的：

- 名称
- 角色
- 怪物
- 地图
- UI
- 音乐
- 音效
- 图标
- 剧情文本
- 数值表
- 具体关卡结构

可以学习的是通用体验：等距视角、ARPG 战斗、装备驱动、地牢循环。

## 第一职业：流亡剑士

定位：近战、位移、爆发、生存。

资源：星尘能量。

职业特色：击杀敌人或命中精英怪后积累星尘能量，用于强化下一次技能。

初始技能：

- 普通攻击：近战基础攻击。
- 冲刺斩：向目标方向突进并造成伤害。
- 星尘爆裂：消耗星尘能量，对周围敌人造成范围伤害。
